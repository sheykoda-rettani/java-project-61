# Проект "игры разума"
В данном проекте реализованы несколько простых интеллектуальных игр
### Доступные игры:
* **Калькулятор** — вычислите результат простой операции.
* **Проверка чётности** — определите, является ли число чётным.
* **Поиск НОД** — найдите наибольший общий делитель двух чисел.
* **Проверка числа на простоту** — проверьте, является ли число простым.
* **Прогрессия** — восстановите пропущенное число в арифметической прогрессии.

### Установка и запуск проекта

Прежде всего установите утилиту `make`. Затем соберите и запустите проект следующей командой:
```shell
  make run 
```

Чтобы повторно запустить проект без пересборки, используйте следующую команду:
```shell
  make run-dist 
```
